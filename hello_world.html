Project While-True-Learn

Fri 12 June 2015

Dear CECian,

Hope you are doing great.

We are embarking on a small but ambitious project to usher in a new culture at CEC. You might be a CS, EC or EEE student who already has many awesome skills under your belt, we would really want you to consider adding one more skill to your skillset.

This skill apparently brings a lot of fear in the mind of many. We are talking about coding.

Computers are built basically for one sole purpose which is to help us with our tasks. Computers are not smart enough to understand what we need right away. We have to give them step by step instructions on what needs to be done. This way of telling computers what needs to be done is basically all what coding is. The world of coding is one of great joy and fun, it fuels everything from the smart phone in your hand to a rover on a Mars mission work.

Steve Jobs had something wonderful to say about learning to code

“I think everybody in this country should learn how to program a computer because it teaches you how to think.”

We know very well how college students hate advices, but this advice from Steve is worth following. You will be thankful 2 or 3 years down the line for having started with coding while at college. It is definitely a skill worth checking out.

Here is what this project plans to offer on a highlevel

  • Make students familiarized with any of the three programming languages ie Python, Java or C.
  • Create a platform where students who are new to programming can get help from mentors who are alumni.
  • Do a small opensource project in Github to give students a flavor of what happens in the real world.

Here are the logistics for the project.

First month

  • Learn either Python, Java or C based on your interest.
  • Learn the basics of version control and get started with github.

Second month

  • Apply what we learned in First month to participate in an opensource project where students will be solving beginner friendly programming problems following a Pull Request review model.

All learning will be a self learning process where we will point to great resources for learning. The idea is not to spoon feed but let students explore as much as you they in any of these tracks. But we want to help you when you are in doubt on how to proceed, so we have mentors who can be communicated through slack.

Finally this is a not for profit but for fun project with a genuine desire to make students interested in programming.To that end it is open to any current CECian irrespective of branch or seniority. The only requirement is a passion to learn and acquire a new skill.

If you are intersted, do register here.

Thanks,

A bunch of CEC Alumni
