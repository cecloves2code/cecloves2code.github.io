Hotspot fix for Linux machines

Thu 15 January 2015

Fixing the hotspot issue

I had a problem with running hotspot on my machine for some time now. I had issues on my old laptop which was running Ubuntu . I had tried many things including the inbuilt option to setup wifi hotspot, even ap-hotspot which appeared to have fixed the issue for many people. I had given up some time back thinking that this was some issue with the drivers and looked at other options like connectify which is a paid application for windows environment.

I formatted my system and installed the latest version of Elementary OS which is awesome btw. So I thought why not try to fix the hotspot issue once again and this time with some good luck i succeeded.

Checking whether your sytem supports Hotspot

We need to ensure that the system you are using actually do support wifi hotspot. 

lspci -k | grep -A 3 -i network

This will give you information regarding the Network controller , adapter , Kernel module etc.

Let's take the Kernel module. For me it was ath5k

modinfo <YOUR_KERNEL_MODULE> | grep depend

You will get few values as result. If you find a value mac80211 , then that means we are good to go as your system will support hotspot.

Installing Dependencies

There are 2 dependencies we might need to install.

a) dnsmasq : This is usually present by default in most of the linux machines.

b) hostapd : We would have to install this seperately.

sudo apt-get install hostapd

Up and Running with hotspotd

hotspotd works as a small daemon which is very easy to use. It is a project hosted in Github and you can report issues if any here.

Installation is pretty straight forward

#Download the latest version from Github
wget https://github.com/prahladyeri/hotspotd/raw/master/dist/hotspotd-0.1.5.tar.gz

#Unzip the file
tar xvf hotspotd-0.1.5.tar.gz

cd hotspotd-0.1.5/

#Compile the setup.py file to install hotspotd
sudo python setup.py install

To run the hotspot

sudo hotspotd start

If you are running for the first time it will ask for set of configurations such as

  • Whether the internet connection is through a USB device such as netsetter (ppo0) or through a wired connection (eth0)
  • A name for the wifi hotspot (SSID)
  • Password for the hotspot

If everything goes well you can see a message

hotspot is running

You can stop the hotspot at any time this way

sudo hotspotd stop
